# Future Skills Roadmap

Este documento lista las skills que planeamos añadir al repositorio para expandir las capacidades de kickstart y desarrollo asistido por IA.

## 1. Implementación & Backend
- **go-hexagonal-standard**: Estándares para microservicios en Go siguiendo arquitectura hexagonal.
- **db-schema-design**: Skill para diseño de esquemas SQL/NoSQL con enfoque en escalabilidad.
- **api-design-first**: Flujo para diseño de APIs usando OpenAPI antes de codificar.

## 2. Operaciones & Observabilidad
- **ai-observability-patterns**: Configuración de logs y tracing optimizados para que agentes de IA puedan depurar errores.
- **docker-standard-bootstrap**: Plantillas de Dockerización para entornos de desarrollo y producción.

## 3. Orquestación de Agentes
- **agent-handoff-protocols**: Convenciones para que múltiples agentes colaboren sin perder contexto.
- **multi-agent-testing**: Cómo testear flujos que involucran varios agentes trabajando en paralelo.

## 4. Calidad & Frontend
- **atomic-design-flutter**: Guía profunda para Atomic Design en Flutter.
- **automated-tui-testing**: Patrones para testear interfaces de terminal (Bubbletea, etc.).
